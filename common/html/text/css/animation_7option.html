css の アニメーション　のオプションは 7つあった！

名前　/ duration / easing / delay / repeat / direction フィルモード

このフィルモードは アニメーションの後のCSSの値をどうするかをしていするもので

none
本来の属性値を適用します（デフォルト）
forwards
アニメーション終了後も最後のキーフレームの値を維持します
backwards
delayの期間中も最初のキーフレームの値を適用します
both
forwardsとbackwardsの両方の効果を持ちます

ということはわざわざjQueryで変更後の値を書き換えたりしなくてよかったのかもしれない

今回はアニメーション後の位置を、アニメーションを指定しているクラスと一緒に記述するか jqueryのcssメソッドで直接書き込むかで対処している。

cssで記述する方がスマートな気がするが、jqueryでその値を取得するとmatrix3dの値に変換された 行列の数字（見てもよく分からない）データを取得してしまうので、jqueryのcssメソッドをつかった。

cssメソッドで指定した値はタグのstyle属性に直接書き込まれて、attrメソッドでそのまま文字列として 取得できるのでこちらの方が操作がしやすかった。

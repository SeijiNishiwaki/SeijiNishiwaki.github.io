セクショニング

認識できないタグは　インラインになる javascriptでは　createElement で　新しいタグを認識させられる


<head>
  <!--[if lt IE 9]>
    <script src="html5shiv.js"></script>
    <style><!-- 認識されたタグの　ブロック要素であるものを　ブロック化する -->
      section,article,aside,nav,header,main,footer,address{
        display:block;
      }
    </style>
  <![endif]-->
</head>

セクショニング系は認識できないブラウザではインライン扱いとなってきけんなので divのidでセクショニングしたものを HTML５でセクショニングタグでラップする感じ

あとから付け足す時にも　こうすれば悪影響はすくないとおもう

セレクタは　タグで選択すると　構造が変わった時に対応できないので セクションに id を振って　その中の必要な要素に クラスを振っていく

			<header>

			</header>
